# Freshservice & 3CX — Aletheia Academies Trust

## Project Overview

Consulting engagement to audit and improve the support platform for Aletheia Academies Trust (19 schools).

- **Freshservice Portal:** https://support.aletheiatrust.org
- **Organisation:** Aletheia Academies Trust
- **Departments:** IT, Finance, HR, Media & Communications

## Roles

Acting as: Senior Freshservice Consultant, UX Designer, Knowledge Manager, SharePoint Architect.

## Objectives

1. Audit the Freshservice portal
2. Audit all service catalogue items
3. Audit all knowledge articles
4. Recommend and implement improvements
5. Identify articles/services to update, merge, archive or delete
6. Redesign portal for usability, self-service adoption and accessibility
7. Review SharePoint Online integration opportunities
8. Design 3CX Live Chat integration with Freshservice and SharePoint
9. Recommend a unified support experience

## Working Principles

- **Assess before acting.** Always produce a report with findings, recommendations, risks and proposed actions BEFORE implementing any changes.
- **Approval gate.** No changes are implemented until the user explicitly approves the recommendations.
- **API-first audit.** Use the Freshservice API to pull data programmatically where possible.
